# Release v9.0.0 — Soft Workbench

AI Launcher Pro v9.0.0 marca a maior reformulação visual do projeto até aqui.

## Destaques

- nova linguagem visual `Soft Workbench`
- interface mais amigável, clara e madura
- presets de cor de destaque no Admin
- nova família oficial de ícones para CLIs e tools built-in
- overrides de ícones com imagem local
- registry central para troca manual futura de assets
- documentação pública atualizada sem prints/mockups antigos

## Mudanças principais

- header, status bar, launcher e superfícies administrativas refeitos
- identidade light-first com dark premium opcional
- nova base de tokens de cor, forma e motion
- centralização da lógica de aparência
- centralização da lógica de ícones built-in

## Observações

- esta release continua local-first e sem telemetria
- os novos prints da v9 serão adicionados depois, em ciclo separado
- para troca manual de ícones, ver `docs/ICON_OVERRIDES.md`

## Descrição curta sugerida para o GitHub

Hub desktop local-first para CLIs de IA, com providers Anthropic-compatible, histórico, custos, presets visuais e customização de ícones.
