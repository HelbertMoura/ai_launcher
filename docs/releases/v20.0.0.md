# AI Launcher v20.0.0 — Command OS

Mega release que transforma o AI Launcher em uma central operacional para agentes: Command Center, inteligência de projeto, Runbooks 2.0, MCP por workspace, Agent Profiles, Sessions 2.0 e melhorias fortes de trust para backup/update/release.

## Highlights

- **Command Center** como tela inicial com workspace ativo, quick launch, readiness cards, sessões e project intelligence.
- **Project Intelligence** detecta stack segura (Node/React/Vite/Tauri/Rust/Python/Go/Docker/MCP) e sugere CLIs/runbooks.
- **`.ailauncher.json` assistido** com preview validado e escrita segura na raiz do projeto.
- **Runbooks 2.0** com presets locais, condições por step e timeline persistida de execução.
- **MCP por projeto** resolve servidores esperados pelo profile, mostra faltantes e resume health.
- **Agent Profiles** para salvar CLI/args/provider por agente e lançar workspace com contexto certo.
- **Sessions 2.0** com métricas, filtros persistidos, replay pelo launch compartilhado e kill confirmado.
- **Backup Trust** com manifest, redaction recursiva de secrets e preview de import.
- **Updater Trust** com cadeia `latest.json`/SHA-256/GitHub Release e auditoria de manifesto.
