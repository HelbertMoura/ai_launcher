# AI Launcher v16.0.1

Patch release de polimento pós-v16 com foco em consistência de launch, UX de Workspaces, QA visual e release hygiene.

## Destaques

- Templates salvos e quick launch agora usam o mesmo fluxo do diálogo principal: provider, workspace env, `.ailauncher.json`, histórico e recent dirs.
- Launcher no modo Vite/browser não exibe mais erro de `invoke` quando roda fora do WebView Tauri.
- Workspaces ganharam confirmação de exclusão e toasts para salvar, importar, exportar e remover.
- Bundle inicial reduzido com code splitting por página: chunk principal caiu de ~634 kB para ~325 kB minificado.
- Empty state da Launcher ganhou ações diretas para adicionar CLI, rodar Doctor e abrir Pré-requisitos.
- Workflow de release limpa artefatos antigos antes do build para evitar assets obsoletos anexados ao GitHub Release.
