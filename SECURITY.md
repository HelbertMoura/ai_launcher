# Security Policy

## Supported Versions

Currently, only the latest release of **AI Launcher** is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 4.0  | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

Security is a top priority for us. If you discover any security-related issues (e.g., vulnerabilities that could allow arbitrary code execution via installed CLI paths or environment variables), please do not disclose them publicly by creating a normal GitHub issue.

Instead, please open a **Private Vulnerability Report** on GitHub:

1. Go to the [Security Advisory page](https://github.com/HelbertMoura/ai_launcher/security/advisories).
2. Click on **Report a vulnerability**.
3. Fill out the form with as much detail as possible to help us reproduce and address the issue quickly.

We will acknowledge receipt of your vulnerability report as soon as possible, and we'll strive to send you regular updates about our progress. If the vulnerability is accepted, we will create a patch, request a CVE (if applicable), and publicly acknowledge your contribution (unless you prefer to remain anonymous).

Thank you for helping keep the open-source community safe!
